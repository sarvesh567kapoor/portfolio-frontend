# Sarvesh Kapoor – Portfolio (React + Vite)

This is a single‑page developer portfolio built with **React**, **Vite** and **Recharts**.

It uses data extracted from Sarvesh Kapoor's CV to showcase:

- Hero section with quick summary
- About section
- Skills & tech stack visualised using bar and pie charts
- Experience cards
- Selected projects grid
- Contact information

## Getting started

```bash
npm install
npm run dev
```

Then open the printed local URL in your browser.

To build for production:

```bash
npm run build
npm run preview
```
